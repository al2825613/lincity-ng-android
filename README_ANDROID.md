# LinCity-NG for Android

هذا المشروع هو عبارة عن هيكل لبناء لعبة LinCity-NG لنظام أندرويد باستخدام SDL2.

## المتطلبات
- Android Studio
- Android NDK
- CMake

## كيفية البناء
1. قم بتحميل هذا المجلد.
2. قم بتشغيل نص الإعداد `setup_android_project.sh` لتحميل كافة المكتبات اللازمة (SDL2, libxml2, fmt, إلخ).
3. افتح المجلد `lincity-android` في Android Studio.
4. انتظر حتى ينتهي Gradle من مزامنة المشروع.
5. قم ببناء المشروع وتشغيله على جهاز أندرويد أو محاكي.

## ملاحظات تقنية
- تم استخدام SDL2 كطبقة وسيطة للرسوميات والصوت.
- تم دمج `libxml2` و `fmt` مباشرة في عملية البناء باستخدام CMake.
- تم نقل ملفات البيانات (data) إلى مجلد `assets` ليتم تضمينها في ملف APK.
- قد تحتاج إلى تعديل بعض أكواد C++ للتعامل مع شاشات اللمس بشكل أفضل.
